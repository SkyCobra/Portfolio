# `og.png` à fournir

Image de preview pour les partages sur réseaux sociaux (Twitter, Discord, LinkedIn, etc.).

**Dimensions recommandées :** 1200 × 630 pixels (ratio 1.91:1)
**Format :** PNG ou JPG (pas SVG — Twitter et plusieurs services ne l'acceptent pas)
**Poids :** < 500 KB idéalement
**Contenu suggéré :** capture du portfolio, logo SkyCobra, ou une bannière dédiée

Déposer le fichier ici sous le nom exact `og.png` (à la racine de `public/`), pas dans un sous-dossier.

Tant que ce fichier n'existe pas, les partages afficheront un placeholder cassé sur certains services ou ignoreront simplement l'image — pas de blocage technique mais perte d'impact visuel.

Outils pour générer rapidement :
- Photoshop / Affinity / GIMP avec un export 1200×630
- Figma
- [og-image.dev](https://og-image.dev) — générateur en ligne
- Claude Design — demander une bannière 1200×630 reprenant le style du site
